# Nanthini — Data Analyst Portfolio (GitHub Pages)

This repository contains a static portfolio site built with Markdown and published via **GitHub Pages**.
Use this repo as the source for your user site: `https://<your-github-username>.github.io`

## How to publish
1. Rename the repository to `SNanthini24.github.io` (or `<yourusername>.github.io`).  
2. Push all files to the `main` branch.  
3. In GitHub settings → Pages, set source to `main` branch and folder `/ (root)`.  
4. Wait a minute and visit `https://SNanthini24.github.io`.

## What to add
- Upload your Jupyter notebooks to `/notebooks/`  
- Upload dashboard `.pbix` files to a safe location and add publish links in `/dashboards/powerbi-links.md` or add screenshots to `/assets/images/`  
- Fill placeholders in files (email, links, pbix filenames)
